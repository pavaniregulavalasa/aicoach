# AI Coach - Telecom Training Platform

AI-powered training platform for telecom systems using LangChain, FastAPI, and Streamlit.

## Requirements

- Python 3.13+
- See `requirements.txt` for all dependencies

## Quick Start

1. Install dependencies: `pip install -r requirements.txt`
2. Configure `.env` file with your API keys
3. Run backend: `uvicorn services.main:app --reload`
4. Run frontend: `streamlit run app.py`

See `LAUNCH_GUIDE.md` for detailed setup instructions.
